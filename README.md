## How This Repo Works
Claude Code agents read files in agents/ and execute
focused tasks. Results are written to CONTACTS.md,
RESEARCH/, OUTREACH.md, and OPPORTUNITIES.md.

Every session starts by reading:
1. README.md (this file)
2. CONTACTS.md (current prospect list)
3. OPPORTUNITIES.md (pipeline status)

Then execute the assigned agent task and commit results.
