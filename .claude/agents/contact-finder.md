# Agent: Contact Finder

## Mission
Find real LinkedIn profiles of analytics and data
engineering leaders at target companies. Write
results to CONTACTS.md and create a RESEARCH/
file for each company.

## Instructions
1. Read CONTACTS.md to see who is already found
2. Read README.md for target company list
3. For each target company not yet researched:
   a. Web search: "[Company] VP Data Analytics
      LinkedIn Charlotte"
   b. Web search: "[Company] Director Data
      Engineering LinkedIn"
   c. Web search: "[Company] Head of Analytics
      LinkedIn"
   d. Find 3-5 real people per company with
      name, title, LinkedIn URL
   e. Note any recent posts or news about their
      data initiatives
4. Write each contact to CONTACTS.md table
5. Create RESEARCH/[company-name].md with:
   - Company overview (size, data team signals)
   - Key contacts found
   - Recent news relevant to data/analytics
   - Personalization angles for outreach
   - Recommended message approach
6. Commit all changes with message:
   "research: add [company] contacts"
