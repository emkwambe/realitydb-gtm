## DNS Security Records (add to mpingo.ai and realitydb.dev)

### SPF Record
Type: TXT
Name: @
Value: v=spf1 include:_spf.google.com ~all
(adjust if not using Google Workspace)

### DMARC Record
Type: TXT
Name: _dmarc
Value: v=DMARC1; p=quarantine; rua=mailto:dmarc@mpingo.ai

### DKIM
Generated by your email provider.
Add the TXT record they provide to DNS.

## Dodo Payments Security Checklist
- [ ] Webhook signature validation implemented
- [ ] Download tokens are UUID v4 (not sequential)
- [ ] Tokens invalidated after first use
- [ ] Rate limiting on download endpoint

## Incident Response
If a token is compromised:
1. Revoke immediately at the issuing service
2. Rotate all tokens that had access to same resources
3. Audit GitHub commit history for unauthorized changes
4. Check npm publish history for unauthorized versions
5. Notify users if any pack was tampered with
6. Document in this file with date and action taken
